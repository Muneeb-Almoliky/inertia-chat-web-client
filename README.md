# Inertia Chat - Web Client

Modern web client for Inertia Chat, built with Next.js.

## Getting Started

### Prerequisites
- Node.js 20+

### Installation
```bash
# Clone repository
git clone https://github.com/Muneeb-Almoliky/inertia-chat-web-client.git

# Install dependencies
npm install

# Start the development server
npm run dev
```
